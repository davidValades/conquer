HTML y CSS desde cero

15 enero 2024 — Instructor: Ana García — Duración: 40 horas

Descripción del curso

En este curso aprenderás a construir páginas web desde cero utilizando HTML5 semántico y CSS3 moderno. Comenzaremos por la estructura básica de un documento HTML y terminaremos maquetando diseños responsive complejos con Flexbox y CSS Grid.

¿Qué aprenderás?

  • Estructura básica de documentos HTML5
  • Etiquetas semánticas: header, nav, main, section, article, footer
  • Formularios accesibles y validación HTML
  • Selectores CSS y especificidad
  • Box model: margin, padding, border
  • Flexbox y CSS Grid para maquetación
  • Diseño responsive con media queries
  • Imágenes responsive con srcset y picture

Requisitos previos

No necesitas conocimientos previos de programación. Solo un ordenador con acceso a internet y muchas ganas de aprender.

Contenido del curso

  2. Tu primer documento HTML
  3. Etiquetas de texto, listas y tablas
  4. Imágenes, audio y vídeo
  5. Formularios en HTML5
  6. Introducción a CSS3
  7. El modelo de caja
  8. Posicionamiento y display
  9. Flexbox paso a paso
  10. CSS Grid
  11. Diseño responsive
  12. Proyecto final: portfolio personal
